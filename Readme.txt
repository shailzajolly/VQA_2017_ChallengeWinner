This file is to run the VQA model of top-down and bottom-up attention. 

The steps to run this code are as follows:

1. First clone or download this repository.

2. Download whole data which contains annotation files, image features and Glove vectors using command:-
bash download_data.sh 
The file will automatically create a data folder in your directory and will add all the files in it. 

3. Once you have all the features and data, it is time to run prepro.py file.
